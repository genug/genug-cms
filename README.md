# genug-cms
a file-based CMS

## status
under development

## requirements
- PHP 7.0
- Apache, .htaccess, rewrite_module (mod_rewrite)

## installation
- Upload all files to the webroot of your web server.

## file encoding
- `.genug` UTF-8 without BOM
- `.page` UTF-8 without BOM
